تقریبا همه توسعه‌دهندگان css از flex برای پیاده‌سازی چیدمان‌های متفاوت استفاده می‌کنند. اما موقعی که از flex استفاده می‌کنیم، ممکن است عرض المان پدر برای جاگیری همه المان‌های فرزند در یک خط کم باشد. در این صورت المان‌های فرزند از پدر خود بیرون میفتند. برای جلوگیری از این مشکل می‌توان از flex-wrap: wrap; استفاده کرد.
